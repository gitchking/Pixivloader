## Quick Start

### Terminal 1 - Backend:
```bash
cd python-backend
python app.py
```

### Terminal 2 - Frontend:
```bash
npm run dev
```

## What Runs:

**Backend:** http://localhost:5000
- API endpoints
- Image download proxy
- Pixiv scraper

**Frontend:** http://localhost:3000
- React UI
- User interface
- Makes requests to backend

## Check if Running:

**Backend health check:**
```bash
curl http://localhost:5000/api/health
```

**Frontend:**
Open browser: http://localhost:3000
